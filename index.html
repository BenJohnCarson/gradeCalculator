Bring up the console to start!

To use...

  • Create a new calculator: calc = new GradeCalculator();
  • To use custom boundaries, pass in a boundaries object: customCalc = new GradeCalculator(customBoundaries);
  • Custom boundaries should be in the following format: { A: 80, B: 70, C: 60, D: 50, E: 40, Fail: "Fail" }
  • Calculate grades from an array: calc.calculateGrade([50,70,89]);
